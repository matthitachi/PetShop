# PET SHOP Platform
## Note
   


PHP 8.3 
Laravel Framework 11.9

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/matthitachi/petshop.git

   ```
2. **Docker setup:**
Setup Docker Compose and run the following command within the project folder:
   ```bash
   docker-compose up -d
   ```
